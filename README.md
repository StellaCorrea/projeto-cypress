# Estrutura do Projeto Cypress 🚀

Este projeto utiliza **Cypress** para automação de testes. Abaixo está a estrutura das pastas e suas funções:

## 📂 Estrutura de Pastas

### 📁 `fixtures/`  
Armazena dados que podem ser usados nos testes, como:  
- **Arquivos JSON**  
- **Mocks**  
- **Outros arquivos auxiliares**  

### 📁 `e2e/`  
Contém os **arquivos de teste**, onde os cenários de automação são escritos.  

### 📁 `support/`  
Armazena arquivos auxiliares e de suporte para os testes, como:  
- **Comandos personalizados (`commands.js`)**  
- **Hooks globais (`e2e.js`)**  
- **Outras configurações compartilhadas**
